# Mirage

A social trading platform built with React, Vite, Tailwind CSS, and TypeScript.

## Features

- **Social Feed** — Post, like, repost, bookmark, and comment
- **Polls** — Create and vote on community polls
- **Trading** — Live positions, candlestick charts, and TradingView integration
- **Marketplace** — Buy and sell usernames and posts with Solana
- **Tipping** — Tip creators directly via Solana wallets
- **Messaging** — Real-time direct messages
- **Notifications** — Stay updated on likes, follows, and replies
- **Explore** — Discover trending hashtags and users

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Lovable Cloud (Supabase)
- **Blockchain:** Solana (tips, marketplace transactions)
- **Animations:** Framer Motion
